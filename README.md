# Calculadora de Promedios

Primera prueba técnica del curso de Python en el Master de Visual Analytics and Big Data.

## Descripción

Esta es una calculadora simple de promedios desarrollada en aproximadamente una hora. El programa permite al usuario:

- Introducir materias y sus calificaciones (notas de 0 a 10)
- Calcular la media general de todas las notas
- Ver qué asignaturas están aprobadas (nota ≥ 5) y cuáles están suspensas
- Identificar la mejor y peor nota obtenida

## Funcionamiento

El programa solicita al usuario que introduzca materias y sus respectivas notas. Una vez finalizada la entrada de datos, muestra:

- Un resumen de todas las materias con sus calificaciones
- La media general de todas las notas
- Lista de asignaturas aprobadas y suspensas
- La mejor y peor nota con su asignatura correspondiente

## Uso

Ejecuta el programa con:

```bash
python calculadora_promedios.py
```

Sigue las instrucciones que aparecen en pantalla para introducir tus materias y notas.
